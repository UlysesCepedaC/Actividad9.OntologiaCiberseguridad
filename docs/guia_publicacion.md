# Guía de publicación

## Publicación del proyecto

1. Se creó un repositorio en GitHub para almacenar la ontología y el agente desarrollado.

2. La ontología desarrollada en Protégé fue exportada en formato OWL/RDF.

3. Se agregaron archivos de documentación, consultas SPARQL y el código del agente en Python.

4. Se realizaron cambios y actualizaciones utilizando Git para llevar control de versiones.

5. El repositorio fue publicado de manera pública para facilitar el acceso y colaboración.
